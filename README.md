# app_for_contest
"ヒリつき"によって起床を促進するbot

## setup
clone repository
```python
git clone https://github.com/ktss1219/app_for_contest
cd app_for_contest
```
set env(Mac)
```python
export "SLACK_BOT_TOKEN" = <ボットトークン>
export "SLACK_APP_TOKEN" = <アプリレベルトークン>
```

download packages
```python
pip install slack_bolt
pip install firebase-admin
```
